# poll-bot

##Overview
A slack poll bot.
This is created by Mini hackthon between friends

##Usage
Open a poll
```/poll open [Title]:[Option 1],[Option 2],[Option 3], ... ```
Cast a vote
```/poll vote [Option Number]```
Close a poll
```/poll close```

##Example
```
/poll open What's our team name?:Ship it!, Git Gud, Works on my machine!
/poll vote 1
/poll close
```
